rustopals::block

Module rustopals::block::pkcs7[−][src]

PKCS7 padding scheme. Pads/unpads plaintext/ciphertext to a multiple of the block size.

The padding byte value (repeated for each byte of padding) is the number of bytes that are added, (i.e. N bytes of value N are added). This implies padding values are <= block_size.

If no padding is needed, a full block (i.e. block_size bytes) of value block_size is added. This is required to disambiguate padding from an actual ending byte. This implies 0x00 padding is invalid.
